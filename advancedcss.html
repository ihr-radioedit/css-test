Advanced CSS Knowledge

What is the difference between `box-sizing: border-box` and `box-sizing: content-box`?

content-box will contain the content in the specified width, but add border and padding to it.
example: width: 400px; padding: 10px; border: 2px; the content is 400px, total width 424px;

border-box will constrain the width to include the specified width plus the border and padding.
example: width: 400px; padding: 10px; border: 2px; the content is 376px, total width 400px;

Describe a situation where you might need to use `!important`.

In most cases, you shouldn't need it. if you do, it's likely because of some selectors with high specificity.

The best use case for it is if a 3rd party style is causing issues with the site layout, using !important would override it.

Describe a situation where you would use display: flex and when you would use display: grid

display: flex is one-dimensional and only allows you to specify direction as a column or a row.

display: grid is two-dimensional and allows you to specify both rows and columns in the same container.

Describe the gap property and when you would use it

in a flex or grid container, gap, row-gap, column-gap allow you to control the space between child elements

What does this selector do? main > div + div { margin-top: 10px; }

> is known as the immediate child selector and only affects divs that are children of main
+ is known as the adjacent sibling selector
adds the margin-top ONLY on any div that is after another div
example, 4 divs inside main, only divs 2,3,4 get the margin, div 1 does not.
