Describe what a css preprocessor does

possible points:

  • compiles css from it's own unique syntax
  • allows nesting
  • can combine multiple files into one
  • allows sharing among multiple files
  • mixins, functions and variables
Explain the uses of the ampersand in SCSS files

known as the parent selector, it can be used in nested styles to combine selectors

a { &:hover { text-decoration: none; } &.red { color: red; } }

that would become

a:hover { text-decoration: none; } a.red { color: red; }

Describe what @include and @extend do

@include is used to add a mixin. it can include arguments if needed

@extend extends styles from one class to another by combining selectors in the compiled css
